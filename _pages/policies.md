## 🛑 ChatGPT and Internet Usage Rules

This exam is open-book. Checking syntax or searching for simple code snippets is permitted.

Copying and pasting entire code from ChatGPT or other sources is prohibited.

If you used ChatGPT:

- You must add the **entire prompt and answer received** as a comment at the bottom of your code file.
- You must be able to understand and explain your own code.
- This allows for plagiarism detection.
