라이브러리 중심으로 배우는 C++ 핵심 과목입니다. C++ 표준 라이브러리를 이용해 함수, 데이터 구조, 클래스, 상속 등 객체지향 프로그래밍 개념을 학습합니다.

This course teaches core C++ with a focus on libraries. Students learn object-oriented programming concepts — functions, data structures, classes, and inheritance — using the C++ standard library.
