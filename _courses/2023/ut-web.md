HTML, CSS, JavaScript의 기본 문법과 웹 표준을 학습합니다. 모듈 1(HTML 구조), 모듈 2(CSS 스타일·레이아웃·반응형), 모듈 3(자바스크립트·DOM)으로 구성되며, 매 주차 실습 과제와 최종 프로젝트를 통해 완성도 있는 웹 페이지를 제작합니다.

This course covers HTML structure, CSS styling and layout, and JavaScript fundamentals — progressing through three modules with weekly practice assignments and a final project.
