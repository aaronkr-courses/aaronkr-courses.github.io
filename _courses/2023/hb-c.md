시간이 지나도 변하지 않는 전문적이고 안전하며 이식 가능한 C 코드 작성법을 학습합니다. 포인터, 동적 메모리 할당, 문자열 처리, I/O, 전처리기, 프로그램 구조, 디버깅 및 테스트까지 C 언어의 고급 기법을 다룹니다.

This course teaches professional, safe, and portable C programming — covering pointers, dynamic memory, strings, I/O, the preprocessor, program structure, and debugging and testing.
